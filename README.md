# 🧘 ZenSpend

**ZenSpend** is a minimalist financial discipline tracker designed to help users distinguish between their *needs* and *wants*. Built with a focus on simplicity and psychological accountability, it rewards responsible spending and tracks "Discipline Points."

## ✨ Key Features

- **Discipline Scoring:** A dynamic score that decreases when you spend on "wants" and stabilizes when you prioritize "needs."
- **Minimalist UI:** A dark-mode, "glassmorphism" design that is easy on the eyes and perfect for mobile use.
- **Real-time Feedback:** Instant visual updates to your discipline bar as you log expenses.
- **Categorized History:** A clean view of recent spending to help you identify patterns of waste.

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS
- **Logic:** Vanilla JavaScript
- **Deployment:** GitHub Pages (Optimized for mobile browsers)

## 🚀 How to Use

1. **Log an Expense:** Enter what you bought and how much it cost.
2. **Classify:** - Tap **Need** for essentials (Rent, Groceries, Bills).
   - Tap **Want** for luxuries (Entertainment, Dining out, Impulse buys).
3. **Monitor:** Keep your Discipline Points in the "Green" zone (70-100). If you hit the "Red" zone, it's time to pause your spending.

## 📱 Mobile Setup

Since this is a single-file application, you can run it directly on your phone:
1. Enable **GitHub Pages** in the repository settings.
2. Open the provided link in your mobile browser.
3. "Add to Home Screen" for a native app-like experience.
